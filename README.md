AI ATS Resume Optimizer

An AI-powered ATS Resume Optimizer built using Spring Boot and React.js that helps job seekers analyze their resumes against job descriptions and improve ATS compatibility.

🚀 Features

->Upload or paste resume content

->Enter job description

->AI-based resume analysis

->ATS compatibility score generation

->Keyword matching and missing skills detection

->Suggestions to improve resume quality

->User-friendly interface

->Backend REST APIs with Spring Boot
